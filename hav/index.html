HAV | HTML Attribute Values

Who?

Aaron Morris

What?

A small github repository and npm package (hav)

hav aims to be the easiest way to work with HTML attributes on the command line.

Why?

  • I have a launch page on my web site. I want command line launch alises driven by it.
  • I have a news page on my web site. I want to open the latest news link from the command line.
  • I want to make command line tools driven by semantic HTML content on the web.
  • I want to encourage people to publish static HTML to the web.
  • I want to use unix tools to do things like "count the links on a web page"

How do I install it?

  • npm install -g hav

How do I use it?

  • curl https://hav.amorris.ca/ | hav .header
  • curl https://hav.amorris.ca/ | hav ^pre text
  • curl https://hav.amorris.ca/ | hav ^h2 name
  • curl https://hav.amorris.ca/ | hav ~why text
  • curl https://hav.amorris.ca/ | hav ~usage text
  • curl https://amorris.ca/launch/ | hav ^a href
  • curl https://amorris.ca/launch/ | hav ^a id
  • curl https://amorris.ca/launch/ | hav music href
